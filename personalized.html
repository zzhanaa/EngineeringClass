Membuat Objek 2D

Video Pembelajaran AutoCAD - Membuat Objek 2D

Rangkuman Materi

Dalam pelajaran ini, kita akan mempelajari:

  1. Menggunakan perintah LINE untuk membuat garis
  2. Membuat lingkaran dengan perintah CIRCLE
  3. Menggunakan perintah RECTANGLE untuk membuat persegi panjang
  4. Teknik snap dan grid untuk presisi
  5. Latihan membuat denah sederhana

Tips: Gunakan shortcut keyboard 'L' untuk Line dan 'C' untuk Circle untuk mempercepat workflow Anda.
